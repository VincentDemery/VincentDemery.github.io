Préliminaires¶

On dfinit ici la longueur et l'nergie gravitationnelle d'une configuration discrtise de la corde $(h_i)$. Le nombre de points de la corde est arbitraire. Avec $N$ points, la distance entre deux points dans la direction $x$ est $\delta x=1/(N-1)$, et la longueur de la corde est donne par $$L=\sum_{i=1}^{N-1} \sqrt{\delta x^2+(h_{i+1}-h_i)^2}.$$

In [3]:
def length(h) :
    N = len(h)
    hc = zeros(N+2)
    hc[1:-1] = h
    dx = 1./(N+1.)
    l = sqrt(dx**2*ones(N+1) + (hc[:-1]-hc[1:])**2)
    return sum(l)

Pour l'nergie gravitationnelle, on procde comme pour la longueur, en prenant la hauteur moyenne de chaque segment : $$E_g=\sum_{i=1}^{N-1} \sqrt{\delta x^2+(h_{i+1}-h_i)^2}\times\frac{h_{i+1}+h_i}{2}.$$

In [4]:
def grav_energy (h) :
    N = len(h)
    hc = zeros(N+2)
    hc[1:-1] = h
    dx = 1./(N+1.)
    hm = (hc[:-1]+hc[1:])/2
    l = sqrt(dx**2*ones(N+1) + (hc[:-1]-hc[1:])**2)
    return sum(l*hm)

def plot_sol (h) :
    N = len(h)
    hc = zeros(N+2)
    hc[1:-1] = h
    dx = 1./(N+1.)
    x = dx * arange(N+2)
    plot(x,hc,'o',label="minimisation de l'nergie")
    return 1

Corde pesante sous tension¶

La forme de la corde pesante sous une tension $T$ est obtenue en minimisant l'nergie gravitationnelle plus une nergie potentielle $E=E_g+T L$.

On utilise la fonction minimize pour minimiser l'nergie de la hauteur discrtise.

In [123]:
tension=1
N=16
sol1 = minimize(lambda x: grav_energy(x) + tension*length(x), zeros(N))

La forme thorique est donne par $$h(x)=A \mathrm{ch}\left(\frac{x-1/2}{A}\right)-T,$$ o $A$ est solution de $$A \mathrm{ch}\left(\frac{1}{2A}\right)=T.$$ On dtermine $A$ l'aide de la fonction brentq .

figsize(8,5)

Corde pesante à longueur imposée¶

La forme de la corde pesante longueur impose est obtenue en minimisant l'nergie gravitationnelle $E_g$ sous la contrainte $L=L_0$.

La fonction minimize avec la mthode 'SLSQP' permet de raliser une minimisation sous contrainte.

In [111]:
L0 = 12
N = 16
h0 = -((L0-1)/2.)*ones(N)
cons = ({'type':'eq', 'fun': lambda x: length(x)-L0})
sol2 = minimize(lambda x: grav_energy(x), h0, method='SLSQP', constraints=cons)

La forme thorique est donne par $$h(x)=A \left[\mathrm{ch}\left(\frac{x-1/2}{A}\right)-\mathrm{ch}\left(\frac{1}{2A}\right)\right],$$ o $A$ est solution de $$2A \mathrm{sh}\left(\frac{1}{2A}\right)=L_0.$$

In [112]:
a = brentq(lambda x: 2*x*sinh(1/(2.*x))-L0, 0.01, 100.)
plot_sol(sol2.x)
X = linspace(0,1)
plot(X,a*(cosh((X-.5)/a)-cosh(.5/a)))

xlabel('x')
ylabel('h(x)')
show()
